# Simple examples of python Script for data engineering

## format_benchmarks_with_pandas

This is a simple script that converts a dataset into different formats (CSV, JSON, Avro, Parquet) and compares the file size and aspeed time.

## chromadb_example 

A example of how use Chromadb with python. 
To run this example you need to install the croma dependencies: `pip install -r requirements_chromadb.txt`


## ingest_weather_forecast

A example to get data from openweathermap.org and store in a parquet file.
